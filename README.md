# Luis M. Portfolio

Welcome to my personal portfolio website! This project showcases my skills as a Front-End Developer, featuring a selection of my best work, including full-stack applications with clean UI and high-performance development.

## Overview

This portfolio is built with Next.js and styled using Tailwind CSS, providing a responsive and visually appealing experience. It includes a main hero section, featured projects, other projects categorized by technology, and a contact form.

## Features

*   **Responsive Design**: Optimized for various screen sizes, from mobile to desktop.
*   **Featured Projects**: Highlights key projects with video previews, descriptions, and tech stacks.
*   **Other Projects**: A comprehensive list of additional projects, sortable by technology, with links to GitHub and live demos.
*   **Contact Form**: Allows visitors to get in touch directly.
*   **Animations**: Utilizes AOS (Animate On Scroll) for engaging scroll-triggered animations and `tw-animate-css` for general CSS animations.
*   **Dynamic Content**: Projects data is managed locally for easy updates.
*   **Visitor Information**: Captures basic visitor IP information (country, continent) and sends it to a Telegram bot for analytics.

## Tech Stack

*   **Framework**: Next.js (App Router)
*   **Language**: JavaScript (with a preference for TypeScript for new files)
*   **Styling**: Tailwind CSS, SASS (`stars.sass` for complex global styles)
*   **UI Components**: shadcn/ui (Button, Card, Tabs, Table, Dialog, Badge)
*   **Icons**: `lucide-react`, `react-icons`
*   **Form Handling**: `@formspree/react`
*   **Animations**: AOS (Animate On Scroll), `tw-animate-css`
*   **Toast Notifications**: `react-toastify`
*   **Backend/API**: Next.js API Routes, Next.js Server Actions
*   **Utilities**: `clsx`, `tailwind-merge` (via `cn` utility)
*   **Telegram Integration**: `node-telegram-bot-api` for notifications.
